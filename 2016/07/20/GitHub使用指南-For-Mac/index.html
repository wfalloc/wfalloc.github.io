Git

Git 是一个分布式版本控制软件。详情 Git 介绍与使用

配置Git

安装完Git后，需要配置用户名称与邮箱

使用 --global 表示在整个系统中使用Git，Git都会默认使用此用户名邮箱作为你的标识。想指定用户名邮箱，不使用 --global 即可。

使用GitHub

注册 GitHub 账号。在提交代码之前需要进行SSH公钥设置。

SSH key设置

1.在本地创建SSH key

your@email.com为你在GitHub上注册账号是使用的邮箱。之后会要求输入密码，成功后会在~/.ssh文件夹下生成公钥和私钥。

2.查看你的public key

3.在GitHub上添加public key

进入GitHub的Settings页

SSH key管理

添加 SSH key
