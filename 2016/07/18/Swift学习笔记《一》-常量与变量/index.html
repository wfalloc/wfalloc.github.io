Swift学习笔记《一》-常量与变量

Swift 中的基本数据类型

Swift 包含了 C 和 Objective-C 上所有的基本数据类型。

整型：Int

浮点型：Float、Double

布尔型：Bool

文本型：String

集合类型：Array、Set、Dictionary

整型

Swift 中整型有位数（8，16，32，64）位之分，有符号（正、负、零）、无符号（正、零）之分。

8位 16位 32位 64位
有符号 Int8 Int16 Int32 Int64
无符号 UInt8 UInt16 UInt32 UInt64

可以通过访问整数类型的 min 和 max 属性来获取对应类型的最小值 最大值。

Swift 中提供一个特殊的整型 Int ， UInt ,其长度会与当前平台系统的位数相同。为了兼容各种位数的系统，提高代码一致性和可复用性。尽量使用 Int ， UInt 类型。

浮点型

Double 64位浮点数。当你需要存储很大或者高精度的浮点数时。
Float 32位浮点数。

布尔型

Bool 取值为 true , false 。

常量和变量

常量：顾名思义，常量的值一旦设定就不能改变。
变量：变量的值可随意改变。

常量和变量的声明

Swift 中使用 let 表示常量， var 表示变量。

同样的你可以定义多个常量或变量。

类型标注

Swift 能够根据你赋的初始值去自行推断常量或变量的类型。同时你也可以指定类型。
